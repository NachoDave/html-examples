Tables

Tables are defined with the <table> element. <th> defines a 'header' data element. There are lots of different border styles that can be defined using the border attribute. width of a cell can be set, % will define width as % of parent element (in this case the body element). To make zebra stripe columns or rows, add :nth-child(even) to the td or tr entry in the stylesheet.

Element Description
<table> Defines a table
<tr> Defines a table row
<th> Defines a header cell in a table
<tf> Defines a data cell in a table
<caption> Put a caption in a table. Use caption-side attribute to specify locatoin
