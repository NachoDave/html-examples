Block level Elements

Block level elements include <p> and <div>. They start on a new line and have space set before and after. They span the width of the page

Div

The div element can be used as a container for other elements. It can be used to style them

Inline level elements

Inline level elements do not start on a a new line and only take up as much space as necessary.

Span inline element

The <span> element can be used to style a section of text in a paragraph
