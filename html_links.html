HTML Links

Links allow connection to other web pages. The attributes can control how links work. The target attribute controls how the page opens

"Standard hyperlink to one of the other pages"
  1. _blank - Opens document in a new tab or window
  2. _self - Default, opens in the same tab or window in which it was clicked
  3. _parent - Opens the documnet in the parent frame
  4. _top - Opens the document in the full body of the window
