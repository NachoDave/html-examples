HTML Attributes

Attributes

  • The src attribute is applied to the <a> to show the source of the image
  • The alt attribute is applied to the <a> alternative text if the image cannot be displayed
  • The style attribute is used to format the design of an element.
  • The lang attribute should be included in the <html> tag to specify the language of the web page
  • The title attribute can be used to add extra info to an element, which can be seen by hovering over the element

W3C recommends making attributes lower case and putting attribute values in double quotes

Attribute Properties

Attributes can have properties, such as font-size
Attribute Property Example
