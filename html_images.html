Images

Use the <img> tag to display an image. THe src attribute gives the image path (in quotations). The alt attribute gives alternative text to be displayed if the the image cannot be displayed.

Image size

The image size can be controlled using the height and width attributes, or the style attribute. The style attribute is used as style="width:128px;height:128px;". Adding the float and specifying left or right attribute to the style allows the image to float to the left or right of text. But got to be honest, it looks rubbish!!

Map

Can use the <map> attribute to make an image with clickable areas. The defines a clickable area in an image map

Favicon

A favicon is a small image displayed next to the image title at the top of a browser tab. It can be added using a <link> with rel attribute = "icon" and type = "image/x-icon" for a .ico file. Can use png, jpg etc
